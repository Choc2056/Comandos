Comando cut en Linux

El comando cut se utiliza para extraer secciones de cada línea de un archivo o de entrada estándar.

Sintaxis básica:

        cut [opciones] [archivo]

Ejemplos de uso:

  • Extraer la primera columna de un archivo CSV:

                    cut -d',' -f1 archivo.csv

    Este comando extraerá la primera columna de un archivo CSV utilizando la coma como delimitador.

  • Extraer un rango de caracteres de cada línea de un archivo de texto:

                    cut -c1-5 archivo.txt

    Este comando extraerá los primeros 5 caracteres de cada línea del archivo de texto.

  • Extraer campos basados en un delimitador personalizado:

                    cut -d':' -f2,3 archivo.txt

    Este comando extraerá el segundo y tercer campo de cada línea del archivo de texto utilizando dos puntos como delimitador.
